# Contributing to SafeGram

Спасибо за интерес к проекту SafeGram! Мы приветствуем вклад от сообщества.

## Как внести вклад

1. **Fork** репозитория
2. Создайте **feature branch** (`git checkout -b feature/AmazingFeature`)
3. **Commit** ваши изменения (`git commit -m 'Add some AmazingFeature'`)
4. **Push** в branch (`git push origin feature/AmazingFeature`)
5. Откройте **Pull Request**

## Стандарты кода

- Следуйте существующему стилю кода
- Добавляйте комментарии для сложной логики
- Пишите понятные commit сообщения
- Тестируйте свои изменения перед отправкой PR

## Структура проекта

- `server-go/` - Backend на Go
- `web/` - Frontend на React/TypeScript
- `desktop/` - Desktop приложение на Electron
- `docs/` - Документация

## Вопросы?

Если у вас есть вопросы, откройте Issue в репозитории.
