
# MVP-Spec (кратко)

- Аккаунты: регистрация (username/email/password/age>=7/phone optional), логин по username/email/phone.
- Чаты: DM / group / channel (sales -> модерация).
- Сообщения: сервер хранит шифротекст (E2EE в DM), realtime через Socket.IO.
- Профиль: displayName, status, bio, links.
- Жалобы и модерация: очередь, approve.
- Боты: каркас + link-preview.
- Темы: light/dark/red&black.
- PIN‑замок UI (web).
