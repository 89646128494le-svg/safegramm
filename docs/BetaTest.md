
# Бета‑тест SafeGram

## Цель
Проверить стабильность ядра (регистрация, логин, чаты, шифрование DM), UX и модерацию.

## Кого зовём
Одноклассники/друзья 12–18+, при желании младше под присмотром.

## Что тестировать (чеклист)
- Регистрация/логин/логаут, ошибки форм.
- Создание DM/группы/канала, канал со «sales» -> попадает в модерацию.
- Установка E2EE‑ключа в DM, отправка/получение сообщений.
- Смена тем (light/dark/red&black).
- Профиль: смена displayName/status/bio/links.
- Жалоба на сообщение -> видна в админке.
- Боты: link‑preview по любой ссылке.

## Куда слать фидбек
Страница **Feedback** в приложении (в шапке), либо /api/feedback (POST).

## Оценка рисков
- Это бета. Возможны падения. Не храните важные данные.
- Медиа‑вложения пока выключены.
