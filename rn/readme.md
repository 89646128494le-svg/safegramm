
# SafeGram RN (заготовка)

Этот каталог — заглушка для будущей Android‑версии на React Native / Expo.

Рекомендации:
- Expo Go для быстрого старта
- Библиотеки: react-navigation, react-native-mmkv (хранение ключей), socket.io-client
- Важно: использовать Android Keystore для шифрования приватного ключа (E2EE)
